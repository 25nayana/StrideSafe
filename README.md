# StrideSafe

StrideSafe is a web application designed to help users navigate safely by providing real-time crime data and location-based route suggestions. The platform uses geolocation to identify safer paths and assist users in making informed travel decisions.

## 🚀 Features
- Real-time crime data visualization
- Location-based safe route suggestions
- Geolocation tracking for current position
- User-friendly and responsive interface
- Interactive maps for better navigation
